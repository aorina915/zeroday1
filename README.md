# zeroday1

A simple web app showcasing React routes, optimized navigation, and a smooth user experience for both visitor and admin flows.

---

## Built With

This project uses React + Vite for fast development and optimized builds.

- Vite for bundling
- React for UI components
- ESLint rules for clean code

Two official React plugins are available:

- `@vitejs/plugin-react` – uses Babel/oxc for Fast Refresh  
- `@vitejs/plugin-react-swc` – uses SWC for Fast Refresh

## React Compiler

Disabled by default due to performance impact.  
See documentation if you want to enable it.
